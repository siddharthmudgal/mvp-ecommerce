# MVP E-commerce



## About

Ecommerce challenge.



## Pre-requisites

Docker desktop



## How to run

#### start containers

```bash
cd mvp-ecommerce
docker-compose -f docker-compose.yml up
```

#### stop containers

```bash
cd mvp-ecommerce
docker-compose -f docker-compose.yml down
```
